Projects

Gitlet

A version-control system designed and written from scratch in Java using object-oriented design that mimics some of the basic features of Git. Functionality includes:

  • Saving the contents of entire file directories.
  • Restoring a version of one or more files or entire commits.
  • Viewing a log.
  • Branching.
  • Merging.

This project was the capstone project for CS 61B, the data structures course at UC Berkeley.
