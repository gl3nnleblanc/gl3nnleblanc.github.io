Projects

QuSP

A playground for simulating 1D quantum systems with matrix product states (also known as tensor trains). There is a fairly comprehensive writeup available at the repository containing further details.

Quantum Partial Search

An implementation of a partial Grover search algorithm using Rigetti's forest API.

Gitlet

A version-control system designed and written from scratch in Java using object-oriented design that mimics some of the basic features of Git.

This project was the capstone project for CS 61B, the data structures course at UC Berkeley.
